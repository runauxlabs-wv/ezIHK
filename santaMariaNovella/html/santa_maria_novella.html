History is the fragrance of the future

1221년 이탈리아 피렌체에서 탄생해 800년 전통을 이어가고 있는 산타 마리아 노벨라는
전통적인 생산방식과 원료, 기술력이 결합하여 탄생한 과거와 현대가 공존하는
프리미엄 라이프 스타일의 브랜드입니다.

산타 마리아 노벨라 800주년을 기념하여 출시된 피렌체 1221 에디션.
800년 헤리티지를 이어가는 새로운 역사의 향기를 소개합니다.

Tabacco Toscano

800년 역사의 영감을 받은 타바코 토스카노의 따듯한 스토리

메디치 가문의 카트리나 공주는
편안한 기분의 도움을 주는타바코 향을 즐겨 사용하여,
타바코는 "Herba catharinaria(카타리나의 허브)"로
불리기도 했습니다.
우디 오리엔탈 노트가 스모키 바닐라 노트와 섞여
신비롭고 부드러운 금빛 향기를 선사합니다.

Top note : 베르가못, 토스카나 타바코 잎
Middle note : 가죽, 앰버, 자작나무
Base note : 시더우드, 유창목, 샌달우드, 머스크, 바닐라

50ml - 1.6 fl.oz. / 100ml - 3.3 fl.oz.

Pot Pourri

800년 역사의 영감을 받은 포푸리의 서정적 스토리

포푸리는 산타 마리아 노벨라의 역사를 함께한 향기로
토스카나 언덕의 꽃잎, 잎사귀, 꽃봉오리가 자아내는
아로마를 재현했습니다.
신선한 월계수와 시트러스가 어우러지는 탑 노트와,
미들 노트의 라벤더, 타임, 로즈마리와 클로브의
아로마틱 노트가 자연의 향기를 전합니다.

Top note : 베르가못, 비터 오렌지, 월계수
Middle note : 라벤더, 타임, 클로브, 로즈마리
Base note : 파출리, 시더우드, 페루 발삼

50ml - 1.6 fl.oz. / 100ml - 3.3 fl.oz.

Melograno

800년 역사의 영감을 받은 멜로그라노의 고혹적인 스토리

진귀한 감각을 불러 일으키는 신비로운 감각 여행을 떠나보세요.
풍요와 번영의 찬송이 펼쳐집니다.
산타 마리아 노벨라의 멜로그라노는 달콤한 플로럴 오리엔탈 노트로,
석류 나무의 풍요와 번영의 의미를 표현하여
따뜻한 환대의 감각을 깨워 줍니다.

Top note : 베르가못, 비터 오렌지, 프레시 스파이스
Middle note : 석류 어코드, 장미, 일랑 일랑
Base note : 오크 모스, 라브다넘, 파출리, 머스크

50ml - 1.6 fl.oz. / 100ml - 3.3 fl.oz.

Acqua della Regina

800년 역사의 영감을 받은 아쿠아 델라 레지나의 로열 스토리

1533년 메디치 가문의 카트리나 공주의 사랑을 담아
프랑스 앙리2세와의 결혼을 축복하며 탄생한 아쿠아 델라 레지나는
산타 마리아 노벨라의 가장 오래된 향기입니다.
아쿠아 디 산타 마리아 노벨라에서
'여왕의 물'을 뜻하는 본래 이름을 되찾아 새로 태어난
프레시 시트러스 부케향을 만나보세요.

Top note : 이탈리안 시트러스 프룻, 네롤리, 페티그레인
Middle note : 네롤리, 로즈마리, 클로브, 라벤더
Base note : 파출리, 머스크

50ml - 1.6 fl.oz. / 100ml - 3.3 fl.oz.

Angeli di Firenze

800년 역사의 영감을 받은 엔젤 디 피렌체의 감동적인 스토리

엔젤 디 피렌체는 1966년 피렌체 대홍수로부터
도시의 심장과 역사를 구해낸 자원봉사자들인
진흙 속 천사들을 위한 향수입니다.
탑 노트의 달콤한 복숭아와 오렌지, 미들 노트의 마린 노트,
블랙 커런트, 자스민이 어우러져 더 달콤한 향을 내는
천사의 향기를 만나보세요.

Top note : 복숭아, 오렌지
Middle note : 마린 노트, 자스민, 블랙커런트
Base note : 샌달우드, 바닐라, 머스크

50ml - 1.6 fl.oz. / 100ml - 3.3 fl.oz.

Rosa Gardenia

800년 역사와 전통의 우아함을 담은 새로운 스토리

꽃의 여왕인 장미와 가데니아의 부드러움이 만나
탄생한 800년 헤리티지의 향기.
탑 노트의 싱그러움, 미들노트의 매혹적인 플로럴과
베이스 노트의 부드러운 연대가 만나
꽃의 향연이 펼쳐집니다.

Top note : 베르가못, 아몬드 블라썸, 오렌지 블라썸
Middle note : 가데니아, 캐비지 로즈, 무화과 꽃, 자스민
Base note : 샌달우드, 바닐라, 머스크

50ml - 1.6 fl.oz. / 100ml - 3.3 fl.oz.

Fresia

800년 역사의 영감을 받은 프리지아의 순수한 스토리

남아프리카에서 기원한 프리지아 꽃은
모든 감각을 깨워주는 매력적인 싱그러움의 아우라를 선사합니다.
산타 마리아 노벨라의 프리지아는 아주 섬세한 향으로,
신비와 미지의 매력을 상징합니다.

Top note : 프리지아 어코드
Middle note : 바이올렛, 센티폴리아 로즈 압솔뤼
Base note : 아이리스, 머스크

50ml - 1.6 fl.oz. / 100ml - 3.3 fl.oz.

Rosa Novella

800년 역사의 영감을 받은 로사 노벨라의 매혹적인 스토리

피렌체 산타 마리아 노벨라 정원에 퍼지는 5월의 향기.
시트러스한 탑 노트가 은은한 장미,
화이트 플라워 향과 어우러지며,
정원의 싱그러움을 오레가노와 페퍼,
그린 허브 노트의 조화로 그려냈습니다.
향으로 펼쳐지는 봄의 정원을 만나보세요.

Top note : 레몬, 페티그레인
Middle note : 캐비지 로즈, 자스민, 가데니아
Base note : 파출리, 샌달우드, 시더우드, 머스크

50ml - 1.6 fl.oz. / 100ml - 3.3 fl.oz.
